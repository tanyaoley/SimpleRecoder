# SimpleRecoder

Функции:


  Запись:
  
  * Начать
  * Остановить
  
  
  Воспроизведение:
  
  
  * Проиграть
  * Пауза
  * Задать позицию воспроизведения(позиция задается тачем)
  * Перемотать на начало
  * Зациклить воспроизведение


Отображается гистограмма звука во время записи и воспроизведения.
